Edit List of Invoices Tab

 

 

Edit List of Invoices tab definition

Title

Description

Invoice Document Number

Display only. The Invoice Number that is undergoing collections.

Billing Date

Display only. The date the invoice was billed.

Billing Period

Display only. The period of time the invoice covered.

The only way to populate the Edit List of Invoices tab is by using a special multiple value lookup called the Look Up / Add Multiple Invoices  to return one or multiple invoices. Use the  button to remove invoices that should not be collected on.
