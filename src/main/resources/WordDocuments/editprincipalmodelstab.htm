Edit Principal Models tab

This tab allows the user to associate one or more security models with the principal.

The display above shows the tab in create mode. In edit mode, the tab presents a display-only set of fields on the left, editable fields on the right in which the user may enter changes, and a delete button at the bottom.

Edit Principal Models tab definition

Title

Description

Model Id

Display only. The unique identifier for the model.

Model Name

Required. The name of a model to apply to the principal. Enter a model name or select it from the lookup .

Member Type Code

Display only. The value is always “Principal”.

Active From Date

The first date on which this model applies to the principal. Enter a date or select it from the calendar . If no date is specified, this model applies to the principal immediately.

Active To Date

The last date (if any) on which this model applies to the principal. Enter a date or select it from the calendar .

add

Click the add button to associate this model with the principal.

delete

Click the delete button to dissociate this model from the principal.
