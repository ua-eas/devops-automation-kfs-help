Position Tab

The Position tab is a display-only area where the attributes of a position are displayed.

 

Position tab definition

Field

Description

Year

The budgeted fiscal year.

Position

Indicates the unique identifier for a position, which represents a set of duties and responsibilities performed by one or more staff or faculty members.

Position Desc

The position description.

Default Object Code

The default object code for the position funding accounting line.

Department Identifier

The chart and organization code.

Job Code Set ID

The job code set ID for the position. This value qualifies the set of job codes.

Job Code

An alphanumeric code representing the job code for the position. Job codes are used to define groups of positions.

Salary Plan

The salary administration plan code for the employee. (Effectively this is the same value as the Position Salary Administration Grade default for the employee's primary position.)

Salary Grade

The salary administration grade code for the employee. (Effectively, this is the same value as the position Salary Administration Grade Default for the employee's primary position.)

Work Months

The number of months worked per year.

Pay Months

The number of months paid per year.

Standard Hours

The standard number of work hours per week.

FTE

 

The proportion of time the position is funded (1.0 = full-time). Position Full Time Equivalency = Position Standard Hours Default/40 * Normal Work Months/Pay Months.

Post Eff Date

The position effective date.
