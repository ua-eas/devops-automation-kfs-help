Organization Options

 >  >  >  >

The Organization Options document is used to set up a 'billing organization' so that users within that organization can create Accounts Receivable documents appropriate to the billing function. This document also allows you to define other data elements that carry forward to the customer invoice, such as payment terms, print options, organization messages appropriate for display on the invoice, remittance information, and the billing organization's phone number(s).
