Organizations Tab

The Organization tab indicates which organizations at your institution are involved in this project. Multiple organizations can be added but only one can be marked as the 'Primary' organization.

Organizations tab definition

Title

 

Description

Chart Of Accounts Code

Required. Enter the code to uniquely identify the Chart of Accounts an organization belongs to, or, search for it from the Chart lookup .

Organization Code

 

Required. Enter the code to uniquely identify the campus, department and school associated with this proposal, or search for it from the Organization lookup .

Name

Display-only. Displays the name of the organization represented by the organization code.

Primary

Select the check box to indicate this is the primary organization sponsoring the award. There can only be one primary organization.

Active Indicator

 

Indicates whether this organization is active or inactive. Remove the check mark to deactivate.

 

Click  to add the new organization to the document.
