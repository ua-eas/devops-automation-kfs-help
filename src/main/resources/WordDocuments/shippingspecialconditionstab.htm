Shipping Special Conditions Tab

The Shipping Special Conditions tab is used to indicate whether Accounts Payable is allowed to pay for additional freight charged when the vendor invoices for goods or services. Examples might include 'Radioactive,' 'Hazmat,' or 'Live Animal.'

Shipping Special Conditions tab definition

Title

 

Description

Shipping Special Conditions

Required. Select the type from the Shipping Special Conditions Type list or select it from the Shipping Special Condition lookup.

Active Indicator

 

Required. Select the check box if the shipping special conditions type is active. Clear the check box if it is inactive.

After selecting a shipping special condition, click  to add it to the document.

   For more information about adding multiple shipping special condition types, see 'Collections'.
