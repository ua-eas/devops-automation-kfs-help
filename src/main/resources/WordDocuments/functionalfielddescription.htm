Functional Field Description

 >  >  >  >  >

The Functional Field Description document is used to maintain descriptions for various fields throughout the application. Note that these values are for data mapping purposes only and cannot actually be viewed elsewhere in the application.

This document can only be initiated by members of the KFS-SYS Manager role and does not route for approval. You can only edit existing field descriptions (by choosing 'edit' from the lookup). Creating new values requires technical assistance.
