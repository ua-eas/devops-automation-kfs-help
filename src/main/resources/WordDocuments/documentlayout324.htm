The Special Circumstances Question document includes the Edit Special Circumstances Question tab.

Edit Special Circumstances Question tab definition

Title

Description

Document Type

 

Required. The document type where this question should display. Select the appropriate value from the list.

     TT – All TEM Document Types (ENT, RELO, TA and TR)

     TRV – All Travel Documents (TA and TR)

     TA – Includes Travel Amendment (TAA)

Question Text

Required. The question that will display on the selected document.

Free Form

Indicates whether (checked) or not (unchecked) the system will display a text box instead of a Yes/No radio button.

Active Indicator

Indicates whether this Special Circumstances Question record is active (checked) or inactive (unchecked).
