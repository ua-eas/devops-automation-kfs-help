Banking Information Tab

Banking Information tab definitions

Title

Description

Bank Applied Date

Display only. Updated when the Travel Manager clicks the  button with the date the button is clicked. This will normally be the date the Travel Manager submits the application to the bank for approval.

Bank Approved Date

Display only. Updated when the Travel Manager approves the CTS Application with the date the document was approved. This will normally be the CTS Card is available for use by the initiator.
