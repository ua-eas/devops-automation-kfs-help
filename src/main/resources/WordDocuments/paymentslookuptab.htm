Payments Lookup Tab

Payments Lookup tab definitions

Title

Description

To view the payment lookup for this Asset

Link (Click here) to the Asset Payment lookup  for this asset.

The only active field on this tab is the Click here link. Clicking on this link displays the Asset Payment Lookup form with the Asset Number field pre-populated with the asset number of the asset being edited here. Below the form, the results table lists all payments processed.

   For more information about the lookup form, see Asset Payment Lookup.
