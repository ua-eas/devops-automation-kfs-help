Payee Section

This section contains information about the person requesting reimbursement for moving or relocation expenses.

Moving and Relocation Overview tab, Payee section definition

Title

Description

Payee Lookup

The name of the person requesting reimbursement (that is, the person to whom reimbursement is to be sent). Use the lookup  to find the person’s name.

     If the initiator is an Arranger, they will need to look up and return a TEM Profile. If the initiator is not an Arranger, the information displayed in this tab is derived from their KIM Person record.

Principal Id

Display only. KIM Principal ID assigned to the TEM Profile Person record.

     This will be blank if the Payee is a non-employee

Payee Type

Display only. Employee or Non-Employee depending on the TEM Profile selected.

Payee Id

Display only. KIM Principal Name assigned to the TEM Profile Person record.

     This will be blank if the Payee is a non-employee

Preparer

Display only. The user who created this document.

First Name

Display only. The first name of the person requesting reimbursement. The system generates this entry from the user’s TEM Profile record.

Last Name

Display only. The last name of the person requesting reimbursement. The system generates this entry from the user’s TEM Profile record.

Address Lookup

If the person uses more than one address, use the lookup  to find the appropriate mailing address for payment.

Street Address Line 1

Display only. The first line of the street address of the payee or the address selected.

Street Address Line 2

Display only. The second line of the street address of the payee or the address selected.

City Name

Display only. The city of the payee or the address selected.

State Code

Display only. The state of the payee or the address selected.

Country Code

Display only. The country of residence of the payee or the address selected.

Zip Code

Display only. The zip code of the payee or the address selected.

Email Address

The email address of the person requesting reimbursement. The system generates the default entry from the user’s TEM Profile record.

Phone Number

The telephone number of the person requesting reimbursement. The system generates the default entry from the user’s TEM Profile record.
