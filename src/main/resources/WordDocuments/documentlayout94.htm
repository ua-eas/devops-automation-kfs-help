Document Layout

The Payment Documentation Location document includes the Edit Documentation Location tab. Selected data fields are available for editing.

Edit Documentation Location tab definition

Title

Description

Documentation Location Code

Required. A code that uniquely identifies the documentation location. The value is often referenced when establishing rules related to documentation location.

Documentation Location Name

Required. The descriptive name of the documentation location. The name appears in the Documentation Location selection box on the DV document.

Documentation Location Address

Required. The full address location where supporting documentation is maintained. This value appears on the DV document cover sheet, when the documentation location is selected.

Active Indicator

Indicates whether this document location code is active (checked) or inactive (unchecked). Remove the check mark to deactivate this code.
