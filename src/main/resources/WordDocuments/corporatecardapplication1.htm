Corporate Card Application 

  >  >>  >

The Corporate Card Application document allows a user with a TEM Profile and default account to request a Corporate Credit Card.

     You must have a TEM Profile with default accounting before you can create this document.

The Corporate Card Application document has four specific tabs: TEM Profile, User Agreement, Department Head Agreement and Banking Information (appears after the Travel Manager clicks ).
