User Agreement Tab

The User Agreement tab allows the initiator to signify their agreement with the responsibilities that come with having a CTS Card issued to them.

The tab has two components:

•      The agreements related to the having a CTS card issued to the initiator.

•      An Accept User Agreement check box. In order to submit this document, this box must be checked.
