Accounts Receivable > Customer and Attribute Maintenance Documents > Collection Event

Document Layout

 

The Collection Event document includes the Edit Collection Event tab. The system automatically enters data into both the Old and New sections. Selected data fields are available for editing.

Edit Collection Activity Event tab definition

Title

Description

Collection Event Code

System generated unique identifying code.

Collection Activity Type

Required. The type of collection activity. Existing collection activity types may be retrieved from the list or from the lookup .

Collection Activity Date

Required. The date the collection activity was performed. Enter a date or use the calendar  to find it.

Collection Activity Comment

Required.  Enter a comment with additional information relating to the collection activity.

Follow-up Date

Required. The date that the follow-up for this collection activity was completed.  This date determines whether or not a collection activity is displayed in the Tickler Report. Enter a date or use the calendar  to find it.

Completed Date

Required. The date that the follow-up for this collection activity was completed.  This date determines whether or not a collection activity is displayed in the Tickler Report. Enter a date or use the calendar  to find it.
