Edit Cost Category Tab

 

Edit Cost Category tab definition

Title

Description

Cost Category Code

A unique code assigned to this cost category.

Cost Category Name

Required. A description for this cost category.

Indirect Cost Indicator

Indicates whether expenditures in this cost category should will be included in the Total Indirect Cost amount on the contracts & grants invoice.  Check the box to indicate if this cost category should be included as indirect cost on the contracts & grants invoice, remove the check mark if this cost category should not be included as indirect cost on the contracts & grants invoice.

Active Indicator

 

Indicates whether this cost category code is active or inactive. Remove the check mark to deactivate this code.
