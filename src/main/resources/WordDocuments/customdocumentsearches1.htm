Financial Processing

Custom Document Searches

 >  >

From Custom Document Searches submenu on the Main Menu tab, users have access to two special search screens related to financial processing.

     Note that these two items are in different locations within the Custom Document Searches submenu. Financial Transactions search is the first item on the submenu, while Disbursement Vouchers search is listed under Financial Processing.

Searches available from the Main Menu, Custom Document Searches submenu

Custom Search

Description

Financial Transactions

Allows you to specify criteria and search for financial transactions of all types. This screen includes additional fields such as account number and total amount to make it easier for you to locate the documents you need.

Disbursement Vouchers

Allows you to specify criteria and search for disbursement vouchers.
