Profile Status Section

Administrator tab, Profile Status section definition

KIM Principal ID

Display only. For a traveler who is an employee, the person’s principal ID. Otherwise, blank.

AR

Customer ID

 

Display only. The AR Customer ID created for this traveler.

    For employees, this will be completed if an advance is issued to the traveler. The AR Customer is created automatically when the Travel Authorization requesting the advance is approved.

    For non-employees, this will be the AR Customer number created in order to create the TEM Profile.

Active

Indicates whether this TEM Profile record is active (Y) or inactive (N).

After entering data for a traveler and card or agency, click  to add it to the document.
