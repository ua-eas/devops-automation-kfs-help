Edit Object Code Tab

In the Edit Account tab, specify the chart and object codes that the sub-object code belongs to. There are two ways to populate the object codes in the Edit Object tab. One is by manually entering or selecting one object code at a time from the normal Object Code lookup , the other is by using a special multiple value lookup called the Look Up / Add Multiple Object Code Lines  to return multiple values.
