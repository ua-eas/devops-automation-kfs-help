Adding One Account and Object Code at a Time

1.    Complete the Edit Global Sub-Object Code tab

2.    To add one object code at a time, enter the values or use the Object Code lookup .

3.    Search for an object code and click the  link associated with the object code you want to select.

4.    Click  to add the object code to the list.

5.    Repeat steps 2 through 4 until you have selected all of the object codes.

6.    To add one account at a time, enter the values or use the Account lookup .

7.    Search for an account and click the  link associated with the account you want to select.

8.    Click  to add the object code to the list.

9.    Repeat steps 2 through 4 until you have selected all of the accounts.
