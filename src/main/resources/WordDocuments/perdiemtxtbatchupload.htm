Per Diem TXT Batch Upload

   For information about the Per Diem XML Batch Upload, see Per Diem XML Batch Upload.

   For information about the Per Diem Batch Processing, see TEM Batch Processes.

   For information about procedures that apply to all KFS batch uploads on the Administration menu, see Batch Upload Basics.

When you select the Per Diem TXT Batch Upload option from the Administration menu tab, the system displays the Per Diem TXT Batch Upload document. This document allows you to upload relational federal per diem data into your institution’s TEM Per Diem tables.

       Federal per diem data is available from http://www.defensetravel.dod.mil/site/perdiemFiles.cfm. This web site provides access to relational input files for both CONUS and OCONUS and Foreign categories. Each relational link provides access to a zip file containing both .xml and .txt files. The .txt file is the preferred input file because it doesn’t take as long to run and it has been more thoroughly tested.
