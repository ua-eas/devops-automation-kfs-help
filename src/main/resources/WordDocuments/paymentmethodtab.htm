Payment Method Tab

The Payment Method tab displays the traveler’s ACH sign-up status.

 

Payment Method tab definition

Title

Description

ACH Sign Up

Display only. Indicates that this traveler is (Yes) or is not (No) signed up for ACH.

ACH Transaction Type

Display only. If the traveler is signed up for ACH, the Pre-Disbursement Processing type will display. Otherwise, None.
