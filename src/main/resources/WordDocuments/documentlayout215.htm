Document Layout

The Labor Object Code Benefits Maintenance document includes the Edit Position Object Benefit tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

     Labor Object Code Benefits Maintenance E-Doc is a Non-Editable Document: You may create a new code for the Labor Object Code Benefits Table, but you may not edit the existing codes since the table contains identifying (key) fields only.

Edit Position Object Benefit tab definition

Title

Description

University Fiscal Year

The fiscal year of the labor benefits type code.

Chart Code

 

The Chart of Accounts code for the labor benefits type code.

Object Code

The salary object code for the labor benefits type code.

Labor Benefits Type Code

A unique code that identifies the labor benefits type code. The types might include 'F' (FICA), 'R' (Retirement), 'G' (Fringe), etc.

Active Indicator

 

Indicates whether this labor benefits type code is active or inactive. Remove the check mark to deactivate this code.
