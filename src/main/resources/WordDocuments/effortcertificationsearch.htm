Effort Certification (search)

 >  >

When you select Effort Certification from the Main Menu, the system displays the Document Lookup screen. Although you may use this screen to search for any type of e-doc, it initially displays fields that are especially helpful in searching for effort certification e-docs.

Effort Certification Document Lookup screen

Title

 

Description

Type

To search for effort certification e-docs, use the default entry of ECD. To search for another type of e-doc, enter the appropriate type code or use the lookup  to find it.

    If you change the Type entry and move the cursor to another field, the system displays fields appropriate to the type of e-doc you have specified.

Initiator

 

To search for effort certification e-docs initiated by a particular user, enter the person's username (principal name) or use the lookup  to find it.

Document/Notification ID

To search for an effort certification based on its unique document number, enter the number.

Date Created From

To search for effort certification e-docs created during a particular period of time, enter the first date in the desired range of dates. You may also use the calendar icon  to select the date. The default is today's date.

Date Created To

To search for effort certification e-docs created during a particular period of time, enter the last date in the desired range of dates. You may also use the calendar icon  to select the date.

Document Description

To search for effort certification e-docs containing a particular description, enter the description.

Organization Document Number

To search for effort certification e-docs containing a particular organization document number, enter the number.

    This number is typically used to identify a related record in another campus system (for example, a customer relationship management system, an order processing system, etc.).  When used in this way, the entry ties a KFS cash control e-doc to a record in the non-Kuali application.

Chart Code

 

To search for effort certification e-docs based on the chart for the source account number(s), select the appropriate chart code from the list or use the lookup  to find it.

Organization Code

 

To search for effort certification e-docs based on the organization code, enter the appropriate code or use the lookup  to find it.

Employee ID

To search for effort certification e-docs associated with a particular employee, enter the appropriate employee ID number or use the lookup  to find it.

Fiscal Year

 

To search for effort certification e-docs for a particular fiscal year, enter the appropriate fiscal year or use the lookup  to find it.

Report Number

To search for effort certification e-docs associated with a particular effort report, enter the appropriate report number or use the lookup  to find it.

Search Result Type

To view the effort certifications in their entirety that match your search criteria, click Document Specific Data. To retrieve only the routing status, date created, and route log for these effort certifications, click Workflow Data.

Name this search (optional)

To save this set of search criteria for later use, enter a name for it.

 

   In many fields you may use special characters to search on a partial entry. For information about using special characters to search, see Data Entry Tips.

   For information about using features and options that are common to all types of custom document searches and working with the search results, see Performing Custom Document Searches.

After you select a specific effort certification e-doc from the search results, the system displays it in view-only mode.

   For information about the fields on an effort certification e-doc, see Document Layout under Effort Certification Document (A21).
