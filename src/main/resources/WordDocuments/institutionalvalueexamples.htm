Institutional Value Examples

•      Grant Description: Inst Values = Institutions currently select from:

o  Federal contract

o  Federal grant

o  Non-federal grant

o  Non-federal, federal pass through grant

o  Non-federal contract

o  Non-federal, federal pass through contract

o  University internal

•      Status: Inst Values = Institutions currently select from:

o  Award (default in Award document)

o  Rejected

o  Underwrite

o  Withdrawn

•      Purpose: Inst Values = the auto-filled value is populated from the selection entered in the Proposal document. Institutions currently select from:

o  Research

o  Instruction

o  Service/other

o  Academic support

o  Student services

o  Scholarships/fellowships

o  Default
