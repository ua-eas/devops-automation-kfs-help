Asset Depreciation Method

 >  >  >  >

The Asset Depreciation Method document defines codes for identifying two depreciation methods (SL - straight-line and SV - salvage value). When SV is used as the depreciation method, the system deducts the salvage amount (the residual amount for which the organization expects to sell the equipment) from the depreciation base amount before calculating depreciation.
