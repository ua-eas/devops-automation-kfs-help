Edit Agency Service Fee tab definition

Title

Description

Agency Service Fee Code

Required. A unique code for this service fee.

     When this code is associated with an imported expense a transfer will be made between the departmantal account and account associated with this code for the amount associated with this code.

Agency Service Fee Description

Description of this code.

Chart Code

 

Required. The institutional chart code where the fee will post.

Account Number

Required. The institutional account number where the fee will post.

Object Code

 

Required. The institutional object code where the fee will post.

Service Fee

The dollar amount of the fee.

Active

Required. Indicates whether this Agency Service Fee record is active (checked) or inactive (unchecked).
