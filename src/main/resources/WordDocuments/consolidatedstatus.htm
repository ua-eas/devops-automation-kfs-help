Consolidated Status

 > >>

   For information about the standard features of KFS reports, see Standard Reports.

The Consolidated Status report shows summary budget, actual, encumbrance and budget balance totals at the consolidation level for each chart, fund, sub fund, and responsibility center. The format is the same as the Consolidated Account Status report except that individual account numbers are not displayed.
